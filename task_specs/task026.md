このタスクは、**青い縦線を境に左右を重ねて、「黒どうしが重なる場所」だけを水色にする**問題です。

入力は 5×7 ですが、中央の青い列は区切りなので捨てます。左側の 5×3 と右側の 5×3 を同じ位置で比較します。

**変換ルール**

左3列を `L`、右3列を `R` とします。

各マスについて、

```text
L のマスが黒(0) かつ R のマスも黒(0) なら、水色(8)
それ以外なら、黒(0)
```

つまり、出力は **左右の黒マスの共通部分** です。

例：

```text
左: 0 0 0
右: 9 0 0
```

この場合、2列目と3列目だけが両方黒なので、

```text
出力: 0 8 8
```

になります。

テスト入力に適用すると、出力は次の形です。

```text
0 0 8
8 0 0
0 0 8
0 0 0
8 0 0
```

一言で言うと、**中央の青線で左右に分け、左右を重ねたときに黒が一致する場所だけを水色で表示する**、というルールです。
