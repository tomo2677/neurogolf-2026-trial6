タスク5の変換は、**「完成している3×3スタンプをテンプレートとして、周囲の未完成スタンプを補完し、その方向へ4マス間隔で繰り返し配置する」** ルールです。入力・出力はいずれも 21×21 です。 

## ルールの言語化

まず、入力内にある **完成形の3×3パターン** を探します。
これは、そのタスク内で最も大きく、3×3枠の中にまとまっている同色パターンです。

この3×3パターンを **スタンプのマスク** として使います。
マスクは回転・反転せず、向きはそのままです。

例えば test[0] では、水色 `8` が完成スタンプです。

```text
8 8 8
8 0 8
8 0 8
```

この完成スタンプの左上を基準位置 `(r0, c0)` とします。
test[0] では、基準スタンプの左上は **row=7, col=6** です。

次に、他の色の小さな断片を見ます。
これらは、完成スタンプと同じ形になるはずの **未完成コピー** です。

断片の位置から、基準スタンプに対する方向を決めます。

* 右にある断片 → 右方向へ複製
* 下にある断片 → 下方向へ複製
* 上右にある断片 → 上右方向へ複製
* 左、上、下左なども同様

スタンプ同士の間隔は、**3×3スタンプ + 1マス空白** なので、上下左右とも **4マス刻み** です。

つまり、方向ベクトルは次のようになります。

```text
左上  (-4, -4)   上  (-4, 0)   右上  (-4, +4)
左    ( 0, -4)                 右    ( 0, +4)
左下  (+4, -4)   下  (+4, 0)   右下  (+4, +4)
```

各断片について、その断片を含む位置に完成スタンプを補完し、さらに同じ方向へ、盤面外に出るまで繰り返し配置します。盤面外にはみ出す場合は、内側に入るセルだけ描きます。

## test[0] への適用

基準スタンプは水色 `8` の3×3パターンです。

```text
8 8 8
8 0 8
8 0 8
```

### 黄色 `4`

黄色は基準スタンプの **右上方向** に断片があります。
なので、同じ3×3マスクを黄色で補完し、右上方向へ4マス刻みで繰り返します。

配置される左上座標は、0-indexed で次です。

```text
(3, 10)
(-1, 14)  ← 上にはみ出すので、盤面内だけ描く
```

### 赤 `2`

赤は基準スタンプの **右方向** に縦線断片があります。
これは完成スタンプの左列部分なので、赤で補完して右方向へ繰り返します。

```text
(7, 10)
(7, 14)
(7, 18)
```

### 緑 `3`

緑は基準スタンプの **下方向** に横線断片があります。
これは完成スタンプの上段部分なので、緑で補完して下方向へ繰り返します。

```text
(11, 6)
(15, 6)
(19, 6)  ← 下にはみ出すので、盤面内だけ描く
```

## 実装向けの整理

擬似コードにすると、かなり小さく書けます。

```python
# 1. 完成している3x3スタンプを探す
#    3x3窓の中で、同じ色の非ゼロセル数が最大のものを基準スタンプにする
seed_color, r0, c0, mask = find_largest_pure_3x3_pattern(input)

# mask は、3x3内で色がある相対座標の集合
# 例: {(0,0), (0,1), (0,2), (1,0), (1,2), (2,0), (2,2)}

out = zeros_like(input)

# 2. 基準スタンプを描く
stamp(out, seed_color, r0, c0, mask)

# 3. 基準スタンプ以外の非ゼロセルから、どの方向の断片かを判定する
directions = set()

for each nonzero cell (r, c, color) not inside seed stamp:
    for dy in [-1, 0, 1]:
        for dx in [-1, 0, 1]:
            if dy == 0 and dx == 0:
                continue

            anchor_r = r0 + 4 * dy
            anchor_c = c0 + 4 * dx

            # そのセルが、隣接する3x3スタンプ枠内の mask 位置に一致するか
            if (r - anchor_r, c - anchor_c) in mask:
                directions.add((color, dy, dx))

# 4. 各方向へ、同じ色でスタンプを繰り返す
for color, dy, dx in directions:
    k = 1
    while stamp位置が少しでも盤面内に入る:
        stamp(out, color, r0 + 4*k*dy, c0 + 4*k*dx, mask)
        k += 1
```

重要なのは、**断片を4連結オブジェクトとして見るのではなく、「3×3スタンプ枠の中にある同色セルの集合」として見る** ことです。train[2] のように、同じ断片が斜めに分かれていて4連結ではない場合でも、同じ3×3枠内なら1つの未完成コピーとして扱います。
