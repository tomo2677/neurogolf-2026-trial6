タスク10の変換ルールは、**灰色の縦棒を「高さ順」に色分けする**処理です。入力・出力はいずれも 9×9 で、入力には黒背景 `0` と灰色 `5` の縦棒だけが出ています。 

## 変換ロジック

入力内の灰色 `5` の各縦棒を、**位置や形はそのまま**にして、灰色から別の色に塗り替えます。

塗り替える色は、縦棒の**高さ、つまりセル数**で決まります。

* 一番高い縦棒 → `1` 青
* 2番目に高い縦棒 → `2` 赤
* 3番目に高い縦棒 → `3` 緑
* 一番低い縦棒 → `4` 黄

黒背景 `0` はそのまま `0` です。縦棒の移動、伸縮、削除、追加はありません。

## 例での確認

train[0] では、灰色の縦棒の高さが以下のようになっています。

* 列6：高さ9 → `1`
* 列2：高さ8 → `2`
* 列4：高さ6 → `3`
* 列8：高さ3 → `4`

そのため、最も高い列6が青 `1`、次に高い列2が赤 `2`、次が緑 `3`、最も低い列8が黄 `4` になります。

train[1] でも同じで、列8が最も高いので `1`、次に列4が `2`、列6が `3`、列2が `4` になります。

## test[0] の場合

test[0] の灰色縦棒は、

* 列2：高さ8
* 列6：高さ7
* 列8：高さ5
* 列4：高さ3

なので、出力では以下になります。

* 列2 → `1`
* 列6 → `2`
* 列8 → `3`
* 列4 → `4`

## コード化向けの表現

1. 出力グリッドをすべて `0` で初期化する。
2. 入力グリッドから、値 `5` を含む列を探す。
3. 各列について `5` の個数を数え、その列の縦棒の高さとする。
4. 高さの大きい順に列を並べる。
5. 並べた順に、色 `1, 2, 3, 4` を割り当てる。
6. 入力で `5` だったセルだけを、割り当てた色に置き換える。

疑似コードにすると：

```python
out = zeros_like(input)

bars = []
for c in range(width):
    rows = [r for r in range(height) if input[r][c] == 5]
    if rows:
        bars.append((len(rows), c, rows))

bars.sort(reverse=True)  # 高さの降順

for color, (_, c, rows) in zip([1, 2, 3, 4], bars):
    for r in rows:
        out[r][c] = color
```

要するに、**灰色の棒グラフを、高い順に 1,2,3,4 の順位色へ変換するタスク**です。
