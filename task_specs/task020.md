## task020 の変換ロジック

このタスクは、**非ゼロ色でできた図形を、同じ色を保ったまま「90度回転対称」に補完する**問題です。
入力と出力はすべて 10x10 で、背景は `0` です。既存の非ゼロセルは変更せず、足りない回転先の `0` セルだけを埋めます。

## ルールの要点

非ゼロセル全体の外接正方形を見つけ、その中心を回転中心にします。
各非ゼロセルを、その中心のまわりに `0度 / 90度 / 180度 / 270度` 回転させ、回転後の位置にも同じ色を置きます。

つまり、各色セルの「90度回転コピー」を追加して、図形全体を4方向回転対称にします。

## 実装手順

座標は JSON 配列基準の **0-indexed** とします。

1. `grid` をコピーして `out` を作る。
2. `grid[r][c] != 0` のセル座標をすべて集める。
3. 非ゼロセルの外接矩形を求める。

```python
r_min = min(r)
r_max = max(r)
c_min = min(c)
c_max = max(c)
```

4. この外接矩形は正方形なので、中心を求める。

```python
cr = (r_min + r_max) // 2
cc = (c_min + c_max) // 2
```

5. 各非ゼロセル `(r, c)`、色 `v` について、中心からの相対座標を取る。

```python
dr = r - cr
dc = c - cc
```

6. そのセルの4回転位置をすべて同じ色 `v` で埋める。

```python
targets = [
    (cr + dr, cc + dc),   # 0度: 元の位置
    (cr + dc, cc - dr),   # 90度回転
    (cr - dr, cc - dc),   # 180度回転
    (cr - dc, cc + dr),   # 270度回転
]
```

7. 各 `targets` がグリッド内なら、`out[nr][nc] = v` とする。
   このタスクでは色の衝突は起きない前提でよいです。既存セルと矛盾するケースはありません。

## 擬似コード

```python
def solve(grid):
    H, W = len(grid), len(grid[0])
    out = [row[:] for row in grid]

    cells = []
    for r in range(H):
        for c in range(W):
            if grid[r][c] != 0:
                cells.append((r, c, grid[r][c]))

    if not cells:
        return out

    r_min = min(r for r, c, v in cells)
    r_max = max(r for r, c, v in cells)
    c_min = min(c for r, c, v in cells)
    c_max = max(c for r, c, v in cells)

    cr = (r_min + r_max) // 2
    cc = (c_min + c_max) // 2

    for r, c, v in cells:
        dr = r - cr
        dc = c - cc

        targets = [
            (cr + dr, cc + dc),
            (cr + dc, cc - dr),
            (cr - dr, cc - dc),
            (cr - dc, cc + dr),
        ]

        for nr, nc in targets:
            if 0 <= nr < H and 0 <= nc < W:
                out[nr][nc] = v

    return out
```

## test[0] での説明

test[0] では、非ゼロセルの外接範囲は `row=2..6`, `col=1..5` なので、中心は `(4, 3)` です。

入力の `(2,3)=4` は中心から見ると `(-2,0)`、つまり「上方向の点」です。これを90度ずつ回転すると、

```text
(2,3) -> 上
(4,5) -> 右
(6,3) -> 下
(4,1) -> 左
```

になるので、出力では `(4,5)=4`, `(6,3)=4`, `(4,1)=4` が追加されます。これにより、色 `4` の配置も90度回転対称になります。

## 実装上の注意

色番号そのものに意味はありません。
`1`, `2`, `3`, `4`, `8` など、どの色でも同じルールで「そのセルの色を回転コピー」します。色ごとの変換表や多数決は不要です。既存の図形を、外接正方形の中心まわりに90度回転対称になるよう補完するだけです。
