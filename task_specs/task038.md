## task038 変換ルール

入力は **9×9**、出力は **1×5**。出力は、入力中に存在する **色 `1` の完全な 2×2 正方形の個数**を、左から `1` で埋める unary 表現にしたものです。色 `2` の 2×2 正方形や、単独の `1` セルは無視します。

---

## 観察されたルール

* `0` は背景。
* `1` が数える対象の色。
* `2` は妨害色で、数えない。
* 入力全体を左上から右下まで走査し、次の形だけを数える。

```text
1 1
1 1
```

つまり、任意の位置 `(r, c)` について、以下がすべて `1` なら 1 個としてカウントします。

```text
input[r][c]     == 1
input[r][c+1]   == 1
input[r+1][c]   == 1
input[r+1][c+1] == 1
```

走査範囲は `r = 0..7`, `c = 0..7` です。

---

## 出力の作り方

カウントを `k` とすると、出力は長さ 5 の 1 行配列です。

```python
output[0][i] = 1 if i < k else 0
```

例：

```text
k = 2 -> [[1, 1, 0, 0, 0]]
k = 3 -> [[1, 1, 1, 0, 0]]
k = 4 -> [[1, 1, 1, 1, 0]]
k = 5 -> [[1, 1, 1, 1, 1]]
```

---

```text

```

### train[1]

```text

```

### train[2]

色 `1` の 2×2 正方形は 4 個。

```text
出力 = [[1, 1, 1, 1, 0]]
```

### test[0]

色 `1` の 2×2 正方形は 3 個。

該当する 2×2 の左上座標は 0-indexed でおおよそ以下です。

```text
(1, 0)
(3, 5)
(7, 3)
```

したがって出力は、

```text
[[1, 1, 1, 0, 0]]
```

## 実装用擬似コード

```python
def solve(grid):
    count = 0

    for r in range(8):
        for c in range(8):
            if (
                grid[r][c] == 1 and
                grid[r][c + 1] == 1 and
                grid[r + 1][c] == 1 and
                grid[r + 1][c + 1] == 1
            ):
                count += 1

    out = [[0, 0, 0, 0, 0]]
    for i in range(5):
        if i < count:
            out[0][i] = 1

    return out
```

---

## ONNX 化向けの最小ロジック

1. 入力 `x` は shape `(9, 9)`。
2. `x == 1` の boolean/int mask を作る。
3. 2×2 window ごとに AND を取る。
4. 全 window の True 数を合計して `count` にする。
5. `[count > 0, count > 1, count > 2, count > 3, count > 4]` を整数化して shape `(1, 5)` にする。

```text
output = [[
  1 if count >= 1 else 0,
  1 if count >= 2 else 0,
  1 if count >= 3 else 0,
  1 if count >= 4 else 0,
  1 if count >= 5 else 0,
]]
```

これが task038 の変換ルールです。
