## task039 の変換ロジック

このタスクは **入力 10×10、出力 3×3** です。添付JSON上でも train 2件・test 1件があり、いずれも入力は10×10、出力は3×3です。 入出力例は task039.json に含まれており、下記ルールで一致します。

### 結論

**入力内の全ての非0セルを含む最小外接矩形を求め、その外接矩形の左上から 3×3 をそのまま切り出す。**

背景色 `0` は背景として扱うが、切り出した 3×3 内に `0` があればそのまま残す。
色の変換、回転、反転、並べ替えは行わない。

---

## 実装手順

入力グリッドを `I` とする。

1. `I[r][c] != 0` である全セルを探す。
2. その全セルについて、最小行 `r0` と最小列 `c0` を求める。

   * `r0 = min(r)`
   * `c0 = min(c)`
3. 出力 `O` は、入力の `r0, c0` を左上とする 3×3 のコピー。

数式で書くと、

```text
O[i][j] = I[r0 + i][c0 + j]    for i = 0..2, j = 0..2
```

ここで `r0, c0` は、全ての非0セルを含む外接矩形の左上座標。

---

## 注意点

**中心の大きな塊だけを見てはいけない。**
train[1] のように、中央の模様から少し離れた点も存在するため、外接矩形は「全ての非0セル」で決める必要がある。

つまり、連結成分の最大領域ではなく、入力全体にある非0セルすべてを対象にする。

---

## 例での確認

### train[0]

非0セル全体の外接矩形の左上は、0-indexで `(2, 2)`。
そこから 3×3 を切り出す。

```text
入力[2:5][2:5] =
[[0, 0, 7],
 [0, 6, 8],
 [7, 8, 4]]
```

これがそのまま出力。

### train[1]

非0セル全体の外接矩形の左上は `(1, 1)`。
左上の離れた `1` も外接矩形に含める。

```text
入力[1:4][1:4] =
[[1, 0, 0],
 [0, 3, 6],
 [0, 5, 2]]
```

これがそのまま出力。

### test[0]

非0セル全体の外接矩形の左上は `(2, 2)`。
そこから 3×3 を切り出す。

```text
入力[2:5][2:5] =
[[0, 0, 0],
 [0, 4, 4],
 [8, 8, 3]]
```

したがって test[0] の出力は以下。

```text
[[0, 0, 0],
 [0, 4, 4],
 [8, 8, 3]]
```

## 擬似コード

```python
def solve(grid):
    rows = []
    cols = []

    for r in range(len(grid)):
        for c in range(len(grid[0])):
            if grid[r][c] != 0:
                rows.append(r)
                cols.append(c)

    r0 = min(rows)
    c0 = min(cols)

    out = []
    for i in range(3):
        row = []
        for j in range(3):
            row.append(grid[r0 + i][c0 + j])
        out.append(row)

    return out
```

## ONNX化向け要約

入力 `10×10` に対して、

```text
mask = input != 0
r0 = first row index where mask has any True
c0 = first column index where mask has any True
output = input[r0:r0+3, c0:c0+3]
```

だけでよい。色ごとの条件分岐は不要。
